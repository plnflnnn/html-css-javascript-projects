# HTML / CSS / JavaScript Projects

This repository contains a collection of frontend layout projects focused on clean markup,
responsive design, animations, and JavaScript interactions.

The main goal of these projects was to practice and improve frontend fundamentals,
work with build tools, animations, UI interactions, and browser compatibility.

---

## Tech Stack

- HTML5 (semantic markup)
- CSS3 / SCSS
- JavaScript (ES6+)
- jQuery & jQuery plugins
- CSS animations & transitions
- Animation libraries (animate.css, wow.js)
- Gulp, Webpack
- Babel, polyfills
- BrowserSync
- Responsive & cross-browser layout
